# Enrollment App

## Overview
The Enrollment App allows users to update a master spreadsheet with data from multiple Excel files. Designed for simplicity, it includes features such as:
- Selecting a master file and a folder of new files.
- Automatically updating or appending data to the master file.
- Progress tracking with a user-friendly interface.

## Features
- Update Excel files with ease.
- Intuitive graphical user interface (GUI).
- Progress tracking for batch updates.
- Automatic detection and handling of duplicate data.

## Installation
1. Download the installer file: `EnrollmentAppInstaller.exe`.
2. Run the installer and follow the on-screen instructions.
3. Once installed, use the shortcut to launch the app.

## Usage Instructions
1. Launch the app via the desktop or Start menu shortcut.
2. Select the **Master Spreadsheet** and **Folder of New Files**.
3. Click **Update Now** to process the files.
4. The progress bar will indicate the status, and a confirmation message will appear upon completion.

## Uninstallation
1. Open **Control Panel** or **Settings > Apps**.
2. Locate **Enrollment App** in the list of installed programs.
3. Click **Uninstall** and follow the prompts.
